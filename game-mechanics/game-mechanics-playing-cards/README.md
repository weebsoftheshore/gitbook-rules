# Game Mechanics - Playing Cards

#### General Rules:

1. Activating or materializing a card counts as playing that card.
2. Only a cardâ€™s owner can play that card.
3. If an attempted play is illegal, that play is canceled and the game state will revert to the point before the play was initiated.
4. All cards in the Effects Stack are controlled by the player who played them.
