# Game Zones - Memory

#### General Rules:

1. The Memory zone is a private zone owned by a player. Each player has their own Memory zone.

{% hint style="info" %}
The Memory zone is referred to by the term “memory."
{% endhint %}

2. Any card that is sent to memory is sent to its owner’s memory.
3. Cards in a player's memory zone will go back to its owner(s) hand(s) during that player's Recollection Phase.
